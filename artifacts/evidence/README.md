# Evidence Artifacts

This folder holds the derived evidence outputs that sit between raw public files and modeled assumptions.

## Intended contents

- profiling summaries produced by [../../scripts/profile_public_evidence.py](../../scripts/profile_public_evidence.py)
- candidate metrics or priors produced by [../../scripts/build_parameter_candidates.py](../../scripts/build_parameter_candidates.py)
- small derived tables that justify modeled ranges or priors

## Current status

No public source is registered yet, so the checked-in evidence artifacts are empty-state outputs by design.
